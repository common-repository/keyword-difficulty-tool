== Description ==
[Keyword Difficulty Tool Demo:](https://goo.gl/DXU8uV)

Not all keywords are easier to rank, use our Keyword Difficulty Tool to find an easy one to get ranked.

TIP: We suggest you pick a low competition keyword if you run a small business site. 3 word phrases like 'keyword difficulty tool' example is a good choice to pick a less competitive niche.

Note: If the tool stops responding, that means too many queries from Google, you might want to wait another day to check again.

**Plugin Main Features are:** 

1.  The tool is great for finding easier to rank keyword phrases, the competition is based by Alexa global ranking. Sites with higher Alexa Rank means harder to compete with.
2.  Mobile responsive design
3.  Animated report graphic design
4.  Instant keyword search results



== Installation ==
From your WordPress dashboard

1. Visit 'Plugins > Add New'

2. Search for keyword 'Keyword Difficulty Tool'

3. Install the Plugin, then Activate it.

4. Find plugin at the bottom of the left sidebar and Click on "Keyword Difficulty Tool" in Admin menu.

5. Input a keyword phrase to test out the report.
